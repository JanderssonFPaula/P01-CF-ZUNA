Configuração não encontrada

O arquivo .env não está configurado corretamente.

Erro: SUPABASE_URL e/ou SUPABASE_KEY não encontrados.
Como corrigir:

Passo 1: Crie o arquivo .env na raiz do projeto:

SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_KEY=eyJhbGciOiJIUzI1NiIs...
SECRET_KEY=mude-essa-chave-para-algo-seguro

Passo 2: Onde encontrar as credenciais:

  • Acesse supabase.com
  • Abra seu projeto → Settings → API
  • Copie o Project URL e a Publishable Key

Passo 3: Reinicie o servidor:

python app.py
